# Calculator (C)

## 📖 소개
C 언어를 공부하기 위해 만드는 콘솔 계산기 프로젝트입니다.

## 🎯 목표
- C 문법 익히기
- 함수 분리
- switch 문 사용
- 프로젝트 개발 방식 익히기
